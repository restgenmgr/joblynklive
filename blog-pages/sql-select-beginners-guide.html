SQL for Beginners – How to Write Your First SELECT Query

📅 April 5, 2026 · 8 min read

If you're new to databases, SQL is the most important skill. This guide teaches you your first SELECT query.

What is SQL?

SQL stands for Structured Query Language. It lets you ask questions of a database.

The Simplest SELECT Query

SELECT * FROM employees;

Filtering with WHERE

SELECT first_name, last_name, department FROM employees WHERE department = 'IT';

Sorting with ORDER BY

SELECT first_name, salary FROM employees ORDER BY salary DESC;

Limiting Results

SELECT * FROM employees LIMIT 3;
